# Python Calculator (Beginner Project)

This is my first Python project. It is a simple calculator that performs basic arithmetic operations.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Integer Division
- Remainder (Modulo)

## Concepts Used
- User input (`input()`)
- Type conversion (`int()`)
- Arithmetic operators
- Basic program structure

## What I Learned
- How to take input from users
- Difference between integer division and float division
- Writing clean and structured code
- Adding comments for better understanding

## Future Improvements
- Add menu-based operation selection
- Handle division by zero errors
- Improve output formatting using modern methods
